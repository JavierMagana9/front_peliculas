(async) consulta(url, methodopt, bodyopt) → {Promise.<Object>}

Funcion para utilizar fetch de una manera mas limpia en los controladores. Realiza una solicitud HTTP a la URL proporcionada utilizando el método especificado. Si el método es POST o PUT, incluirá el cuerpo de la solicitud en formato JSON. En caso de error en la solicitud, atrapará el error y lo devolverá.
Parameters:
Name Type Attributes Default Description
url string La URL a la que se enviará la solicitud.
method string <optional>
 'GET' El método HTTP que se utilizará para la solicitud.
body Object <optional>
 {} El cuerpo de la solicitud, que se enviará con métodos POST o PUT.
